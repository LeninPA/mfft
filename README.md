# mfft
Multiple Fast Fourier Transform

## FFT

La transformada rápida de fourier es una parte indispensable en el cómputo contemporáneo. Este proyecto consiste en la paralelización de FFT para 3D realizando la FFT para cada rebanada de la matriz usando OpenMP.

### Opción 1
> Completar el esqueleto del código de C (utilizando la implementación de FFT escrita en fortran, i.e.: sólo es paralelizar la distribución de las tareas)

### Opción 2
> Traducir la solución del código (implementar el kerner de FFT, en este caso es posible sólo escribir el kernel)
